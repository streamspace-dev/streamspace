# Advanced Audit Logging Plugin

Enhanced audit logging with search, export, retention, and compliance reports.

## Features
- Comprehensive audit trail
- Advanced search and filtering
- Export to CSV/JSON
- Retention policies
- Compliance reporting

## Installation
Admin → Plugins → "Advanced Audit Logging" → Install

## Configuration
```json
{"enabled": true, "retentionDays": 2555, "logLevel": "detailed"}
```
